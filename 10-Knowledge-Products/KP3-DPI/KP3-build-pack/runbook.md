# KP3 build pack — run book

Stand this slice up from zero on the Linkup (X-Road) + Joget stack.

## Prerequisites
- KP2 build pack deployed (it is the layer below).
- ITU cloud access live (Linkup + Joget DX 8.x) — Inception Report action item A4.

## Steps
1. Deploy the configs:  `scripts/deploy.*`
2. Seed Progressa demo data:  `scripts/seed.*`
3. Run the acceptance suite:  `scripts/acceptance.*`
